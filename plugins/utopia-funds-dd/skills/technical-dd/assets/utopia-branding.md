## Typography

- **Primary font**: Arial

- **Body**: Arial Regular, 12pt
- **Table text**: Arial, 10pt
- **Header/Footer**: Arial, 8pt

## Logo

The Studio logo should be provided by the user as an upload, or referenced if already available in the project. If no logo is available, use text-only branding: "The Studio — Utopia Capital" in Deep Navy, Arial Bold 14pt.

## Classification Marking

All reports carry: "Strictly Confidential"
- Font: Arial Bold, 11pt
- Color: #C0392B (red)
- Appears on cover page and can be included in header

## Footer Standard

Left: "The Studio — Utopia Capital"
Right: "Page [X]"

## Overriding Branding

If the user provides custom brand assets:
- Ask for: logo file, primary accent color hex, secondary color hex, font preference
- Replace the corresponding values above
- Keep the evidence hierarchy colors unchanged (these are standardized across all reports)
