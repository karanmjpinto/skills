# Pitch Deck Scoring Framework

Score every deck twice — once on the source materials (what you're given to work with) and once on the output deck (what you produce). This creates a clear before/after picture and highlights where the skill added the most value.

---

## How to Use This Framework

1. **Score the input materials** (1–10) on each dimension BEFORE generating the deck
2. **Generate the deck** using the skill's full workflow
3. **Score the output deck** (1–10) on the same dimensions
4. **Report the delta** — where did the skill add the most value, and where are gaps the user must fill?

Present scores as a summary table at the end of deck generation.

---

## Scoring Dimensions (15 dimensions, 1–10 each)

### Dimension 1: Narrative Clarity
*Does the deck tell a single, coherent story from first slide to last?*

| Score | Description |
|-------|-------------|
| 1–3 | No clear story; slides feel like disconnected facts. No logical flow. |
| 4–5 | A rough story exists but has gaps or jumps. Some slides don't connect to adjacent ones. |
| 6–7 | Clear narrative arc from problem to ask. Most slides logically follow. Minor transitions feel forced. |
| 8–9 | Strong, compelling story. Every slide advances the argument. Transitions feel natural. Investor can retell the story after reading. |
| 10 | Exceptional narrative. The Global Context sets up the world, problem-solution delivers the thesis, flywheel proves compounding, ask feels inevitable. Nothing extraneous. |

### Dimension 2: Problem Definition
*Is the problem specific, urgent, and felt — not just described?*

| Score | Description |
|-------|-------------|
| 1–3 | Problem is vague, generic, or unsubstantiated. Could apply to any company. |
| 4–5 | Problem is stated but not made vivid. No data. Feels abstract. |
| 6–7 | Clear problem with some data or specificity. Investor understands the pain intellectually. |
| 8–9 | Specific, urgent, data-backed problem. Investor can feel the pain. Current solutions clearly shown to be inadequate. |
| 10 | Visceral. The problem is undeniable, quantified, and the investor immediately thinks "someone needs to fix this." |

### Dimension 3: Solution Strength
*Does the solution clearly and directly address the stated problem?*

| Score | Description |
|-------|-------------|
| 1–3 | Solution is unclear or doesn't connect to the problem. Feature dump with no benefit framing. |
| 4–5 | Solution addresses the problem but is vaguely described. Benefits not clearly stated. |
| 6–7 | Clear solution with benefits mapped to the problem. Differentiation starting to emerge. |
| 8–9 | Strong value proposition. Direct mapping to problem. Clear differentiation. Benefits-first language. |
| 10 | Solution feels inevitable given the problem. Differentiation is structural, not just feature-based. Use cases are concrete and compelling. |

### Dimension 4: Market Sizing Credibility
*Is the market opportunity real, big enough, and believable?*

| Score | Description |
|-------|-------------|
| 1–3 | No market data, or only a vague top-down TAM with no methodology. |
| 4–5 | Some market data but sources unclear. Top-down only. No customer profile. |
| 6–7 | TAM/SAM/SOM with methodology. At least one credible source. Customer profile exists. |
| 8–9 | Bottom-up SAM calculation with cited sources. Growth drivers explained. Customer avatar is specific. |
| 10 | Rigorous bottom-up sizing with multiple credible sources. Growth trends validated by research. SAM >$1B with clear logic. |

### Dimension 5: Traction Evidence
*Is there proof that the business is working?*

| Score | Description |
|-------|-------------|
| 1–3 | No traction data. Or only projections with no actuals. |
| 4–5 | Some metrics but no trend lines. Static snapshot. Vanity metrics dominate. |
| 6–7 | Meaningful metrics with time dimension. Growth visible but not contextualized. |
| 8–9 | Strong metrics with clear growth trajectory. Stage-appropriate KPIs. Customer proof (logos, testimonials). Retention visible. |
| 10 | Transparent, all key metrics on display. Hockey stick growth or clear inflection point. Cohort data. Investors can calculate unit economics from the slide. |

### Dimension 6: Competitive Positioning
*Does the deck demonstrate deep market knowledge and a defensible position?*

| Score | Description |
|-------|-------------|
| 1–3 | No competitive analysis. Or claims "no competition." |
| 4–5 | Competitors listed but no positioning framework. Generic feature comparison. |
| 6–7 | 2×2 matrix or comparison table with reasonable axes. 4–6 competitors. Position looks credible. |
| 8–9 | Thoughtful positioning with axes that reflect structural market dynamics. Honest about competitor strengths. Differentiation is clear and defensible. |
| 10 | Competition slide reveals deep market insight. Axes could not be replicated by a competitor. Positioning connects to the Global Context forces. Investor thinks "these founders really know their market." |

### Dimension 7: Business Model Clarity
*Can the investor understand how money is made in 30 seconds?*

| Score | Description |
|-------|-------------|
| 1–3 | Revenue model unclear or missing. No unit economics. |
| 4–5 | Revenue model stated but pricing unclear. No unit economics. |
| 6–7 | Clear revenue streams and pricing. Some unit economics (CAC or LTV but not both). |
| 8–9 | Revenue model, pricing, and full unit economics (LTV, CAC, margins, payback). Revenue math is calculable. |
| 10 | Revenue engine is crystal clear. Unit economics are strong. Path from current model to scale economics is visible. |

### Dimension 8: Team Credibility
*Does this team have the right people to execute?*

| Score | Description |
|-------|-------------|
| 1–3 | No team information. Or team bios are generic with no domain relevance. |
| 4–5 | Team listed but connection to the problem/market is weak. Key gaps not addressed. |
| 6–7 | Relevant team with some domain expertise. Key roles filled. Some advisors. |
| 8–9 | Strong team with clear domain expertise. Previous startup/exit experience. Advisors add credibility. Gaps acknowledged with hiring plan. |
| 10 | Team is a competitive advantage. Deep domain expertise, relevant exits, complementary skills. Advisors are industry heavyweights. Investor thinks "if anyone can do this, it's this team." |

### Dimension 9: Financial Realism
*Are the projections credible and the assumptions transparent?*

| Score | Description |
|-------|-------------|
| 1–3 | No financials. Or hockey stick projections with no justification. |
| 4–5 | Some projections but assumptions hidden. Inconsistent with current traction. |
| 6–7 | 12–36 month projections with stated assumptions. Mostly consistent with traction. Burn rate visible. |
| 8–9 | Clear projections with explicit assumptions. Path to profitability (or next milestone). Burn and runway stated. Scenarios considered. |
| 10 | Financials are a strength. Assumptions are testable. Unit economics support the projections. Investor can stress-test the model from the slide. |

### Dimension 10: The Ask Precision
*Is the funding request specific, justified, and connected to milestones?*

| Score | Description |
|-------|-------------|
| 1–3 | No specific ask. Or a wide range with no use of funds. |
| 4–5 | Amount stated but use of funds vague. No milestones tied to the capital. |
| 6–7 | Specific amount with high-level use of funds. Some milestones. |
| 8–9 | Specific amount, detailed use of funds, clear milestones this capital achieves. Timeline stated. |
| 10 | Ask feels inevitable after the rest of the deck. Specific amount, detailed allocation, milestones that de-risk the next round. Investor knows exactly what their money buys. |

### Dimension 11: Global Context Quality (Studio Signature)
*Do the three forces pass the four tests? Is this slide intellectually compelling?*

| Score | Description |
|-------|-------------|
| 1–3 | No global context. Or forces are just buzzwords / trends. |
| 4–5 | Forces exist but feel like "why now" trends, not structural truths. One or more fail the four tests. |
| 6–7 | Three distinct forces from different domains. Most pass the four tests. Anchor stats present but could be stronger. |
| 8–9 | All three forces are directional, irreversible, structural, and privately connected. Anchor stats are specific and citable. Differentiator labels are subtle. |
| 10 | Investor reads this slide and nods — "yes, obviously." Forces feel like physics, not opinion. The implicit connection to the company clicks later and creates an "aha" moment. Self-audit in speaker notes shows rigorous thinking. |

### Dimension 12: Growth Flywheel Quality (Studio Signature)
*Does the flywheel show real compounding mechanics with evidence?*

| Score | Description |
|-------|-------------|
| 1–3 | No flywheel. Or a linear process flow mislabeled as a flywheel. |
| 4–5 | Circular diagram exists but moats are aspirational with no evidence. Feels theoretical. |
| 6–7 | Clear flywheel with 3–5 nodes. Some moats have evidence. Key metric present but not strongly defended. |
| 8–9 | Flywheel is compelling. Moats are labeled as real vs. aspirational. Evidence for at least 2 moats. Key metric proves the flywheel is turning. Connects to GTM. |
| 10 | Flywheel is the most memorable slide in the deck. Every moat has evidence. Key metric is strong. Speaker notes address what could break it (intellectual honesty). Forces from Global Context map to flywheel moats. |

### Dimension 13: Visual Design & Professionalism
*Does the deck look like it was made by a company worth investing in?*

| Score | Description |
|-------|-------------|
| 1–3 | Text-only slides. No visual hierarchy. Inconsistent formatting. |
| 4–5 | Some visual elements but generic. Inconsistent styling. Low contrast or readability issues. |
| 6–7 | Clean, consistent design. Brand colors applied. Layout variety. Visual elements on most slides. |
| 8–9 | Professional, polished design. Strong visual hierarchy. Charts and diagrams reinforce content. No slide repeats the same layout. |
| 10 | Design is a competitive advantage. Every slide has a visual element that strengthens the message. Layout variety is natural, not forced. The deck feels premium without being overproduced. |

### Dimension 14: Data Integrity
*Is every claim backed by evidence? Are sources cited?*

| Score | Description |
|-------|-------------|
| 1–3 | Claims made with no data. Or obviously fabricated numbers. |
| 4–5 | Some data but sources unclear. Mix of sourced and unsourced claims. |
| 6–7 | Most claims have data. Key sources cited. A few gaps. |
| 8–9 | All major claims backed by data from source materials or research. Sources cited. Discrepancies flagged. |
| 10 | Every claim is traceable. Cross-referenced where possible. [NEEDS INPUT] placeholders used honestly where data is missing. Investor can verify any number. |

### Dimension 15: Audience Calibration
*Is the deck written for the right audience with the right emphasis?*

| Score | Description |
|-------|-------------|
| 1–3 | No clear audience. Language and emphasis feel generic. |
| 4–5 | Audience implied but emphasis doesn't match their priorities. |
| 6–7 | Written for the stated audience. Language mostly appropriate. Some misplaced emphasis. |
| 8–9 | Strong audience awareness. Co-investor decks emphasize returns and risk; founder coaching decks emphasize vision and traction. Industry-specific metrics included. |
| 10 | Deck feels tailor-made for the audience. Every word choice, metric selection, and emphasis reflects what this specific audience cares about. |

### Grading Scale

| Grade | Score Range | Meaning |
|-------|------------|---------|
| **A** | 120–150 | Investor-ready. Minor polish only. |
| **B** | 90–119 | Strong deck with specific areas to strengthen. Ready for warm intros. |
| **C** | 60–89 | Solid foundation but significant gaps. Needs another pass before sending. |
| **D** | 30–59 | Major structural issues. Requires substantial rework. |
| **F** | 0–29 | Not ready. Source materials too sparse to build a credible deck. |

### After Scoring

For any dimension scoring 6 or below, provide:
- **What's missing**: Specific gap identified
- **What to do**: Concrete action the user can take (provide data, answer a question, approve a research finding)
- **Impact if fixed**: How many points this dimension could gain

For the Global Context and Flywheel dimensions specifically, if they score below 7, reference `slide-frameworks.md` and offer to regenerate those slides with more targeted research.
