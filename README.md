# ⚠️ This repo has moved

The Utopia skills marketplace is now maintained at:

## 👉 [**The-Utopia-Studio/skills**](https://github.com/The-Utopia-Studio/skills)

All skills, packs, and updates live there now. This repo is kept as a mirror so existing Claude Code installs don't break, but **all future updates happen in the new location**.

---

## For new users

Register the new marketplace instead:

```
/plugin marketplace add The-Utopia-Studio/skills
```

Then install packs as usual:

```
/plugin install utopia-studio-cobuild-discovery@skills
/plugin install utopia-funds-dd@skills
```

See the [Team Guide](https://github.com/The-Utopia-Studio/skills/blob/main/GUIDE.md) for the full walkthrough.

---

## For existing users who registered `karanmjpinto/skills`

Your current setup still works — this repo is mirrored. But to get future updates, switch to the new marketplace once:

```
/plugin marketplace remove karanmjpinto/skills
/plugin marketplace add The-Utopia-Studio/skills
```

Then reinstall whichever packs you use.

---

## Why did this move?

Skills now live in [The Utopia Studio](https://github.com/The-Utopia-Studio) org so the whole Studio team and Funds team can co-own and contribute. This repo was a one-person sandbox — the new home makes it a shared resource.
