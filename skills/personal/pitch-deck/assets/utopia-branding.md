# The Studio — Utopia Capital | Pitch Deck Branding

Default branding for Studio pitch decks. When building decks for portfolio companies, ask the user for company-specific brand assets and override these values.

---

## Studio Identity

- **Entity name**: The Studio — Utopia Capital
- **Short name**: The Studio
- **Confidentiality**: "Strictly Confidential" on all co-investor decks

## Studio Color Palette

| Role | Color | Hex |
|------|-------|-----|
| **Primary / Dark backgrounds** | Deep Navy | #1B2A4A |
| **Secondary / Accent** | Steel Blue | #2E75B6 |
| **Highlight / Callout** | Gold | #D4A843 |
| **Body text (light bg)** | Near Black | #1A1A2E |
| **Subtitle / secondary text** | Dark Gray | #333333 |
| **Muted / labels** | Medium Gray | #666666 |
| **Light backgrounds** | Off White | #F5F5F5 |
| **White** | White | #FFFFFF |

### Slide Background Strategy

- **Title slide**: Deep Navy background, white text, Gold accent
- **Global Context + The Ask**: Deep Navy background (bookend the narrative)
- **Content slides**: White or Off White background, dark text
- **Section dividers** (if needed): Steel Blue background, white text

## Typography

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Slide title | Arial | 36–44pt | Bold |
| Subtitle / tagline | Arial | 20–24pt | Regular |
| Body text | Arial | 14–16pt | Regular |
| Bullet points | Arial | 14–16pt | Regular |
| Stat callouts (large numbers) | Arial | 48–72pt | Bold |
| Stat labels | Arial | 12–14pt | Regular |
| Captions / sources | Arial | 10–12pt | Regular, Medium Gray |
| Footer | Arial | 8pt | Regular |
| Speaker notes | Arial | 12pt | Regular |

## Footer

- Left: "The Studio — Utopia Capital"
- Right: "Strictly Confidential"
- Color: Medium Gray (#666666) on light slides, White (#FFFFFF) on dark slides

## Logo

If available, place the Studio logo on the title slide (left-aligned) and optionally in the footer. If no logo file is available, use text branding: "The Studio — Utopia Capital" in Deep Navy, Arial Bold 14pt.

---

## Portfolio Company Override

When building a deck for a portfolio company (founder coaching mode):

1. **Ask for**: Company logo, primary brand color, secondary color, preferred font
2. **Replace**: All Studio branding with company branding
3. **Remove**: "The Studio" footer and confidentiality marking
4. **Keep**: The slide structure, frameworks, and content quality standards

If the company has no defined brand:
- Use a clean neutral palette: Charcoal (#36454F) primary, Off-White (#F2F2F2) backgrounds, a single accent color derived from their logo or website
- Font: Arial (safe default)
